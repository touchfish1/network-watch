Web 端

桌面主节点默认启动 Web 服务（可通过环境变量控制）。你可以用浏览器访问，或在局域网中用手机/平板查看。 
默认地址：http://<主机IP>:17321/
健康检查：GET /api/v1/health
快照：GET /api/v1/snapshot
集群：GET /api/v1/machines

Agent（无 GUI Linux）

适用于服务器、软路由、树莓派等无图形环境。Agent 会采集系统指标，并自动发现局域网的 GUI 主节点上报。 
# 运行时环境变量（示例）
NETWORK_WATCH_MACHINE_ID=linux-node-01
NETWORK_WATCH_DISCOVERY_PORT=17322
NETWORK_WATCH_DISCOVERY_INTERVAL_SECS=10
NETWORK_WATCH_NODE_TTL_SECS=30
NETWORK_WATCH_CAPABILITY_PATH=/api/v1/capabilities
# 兜底：发现为空时上报到固定主节点
NETWORK_WATCH_COLLECTOR=http://192.168.1.100:17321/api/v1/ingest

隐私与安全

Network Watch 默认只在本机/局域网工作。若你在不可信网络使用，请将 Web 绑定到 127.0.0.1， 或在反向代理层加鉴权（后续也可在程序内置 token）。 
NETWORK_WATCH_WEB_BIND=127.0.0.1:17321
